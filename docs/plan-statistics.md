# 통계 메뉴 구축 계획

> 최초 작성: 2026-03-24
> 상태: 기획 중

---

## 메뉴 구조

```
통계
├── 도정 실적 분석   (수율 중심)
├── 농가/재고 현황   (미처리 재고 파악)
└── 포장/출고 현황   (생산·출고 대비)
```

---

## 공통 UX 패턴

모든 통계 페이지는 아래 구조를 공유한다.

- **상단:** 요약 카드 3~4개 (핵심 숫자 한눈에)
- **중단:** 차트 (탭으로 보기 기준 전환)
- **하단:** 상세 테이블 (드릴다운용)

---

## 페이지 1: 도정 실적 분석

- **경로:** `/statistics/milling`
- **상태:** `[ ]` 기획 / `[ ]` 개발 / `[ ]` 완료

### 사용 데이터
- `MillingBatch` + `Stock` + `MillingOutputPackage`

### 필터 UI 구조
```
[기간별]  [품종별]  [도정구분별]                   ← 상단 탭 (기간별 기본)
  ↓ 기간별 선택 시
[2025년산] [1년] [6개월] [3개월] [1개월] [1주] [📅]  ← 빠른기간 칩 (1개월 기본)
[품종 ▼]  [도정구분 ▼]                             ← 보조 멀티셀렉트 필터
```

- `📅` 클릭 시 날짜 직접 입력 인라인 확장
- 품종·도정구분 미선택 시 전체 포함

### 빠른기간 → 집계단위 자동 매핑
| 버튼 | 기간 | X축 단위 |
|------|------|---------|
| 1주 | 7일 | 일별 (`MM/DD`) |
| 1개월 | 30일 | 주별 (`MM/DD~`) |
| 3개월 | 90일 | 주별 |
| 6개월 | 180일 | 월별 (`YYYY-MM`) |
| 1년 | 365일 | 월별 |
| 연산 | 해당 연산 전체 | 월별 |
| 직접입력 | 선택값 | ≤14일→일별 / ≤90일→주별 / 91일+→월별 |

### "연산(cropYear)" 필터 처리
- `Stock.productionYear` 기준 (`MillingBatch → stocks → productionYear`)
- 날짜 범위: 해당 연산 배치들의 실제 date 범위 사용

### Server Action 파라미터
```ts
type MillingStatsParams = {
  from: Date
  to: Date
  groupBy: 'day' | 'week' | 'month'
  varieties?: string[]
  millingTypes?: string[]
  cropYear?: number          // 설정 시 날짜 필터 대체
}
```

### 요약 카드 (4개, 차트 우측 세로 배열)
| 카드 | 계산 | 색상 |
|------|------|------|
| 총 투입량 (kg) | `MillingBatch.totalInputKg` 합계 | 파랑 `#00a2e8` |
| 총 생산량 (kg) | `MillingOutputPackage.totalWeight` 합계 | 초록 `#8dc540` |
| 평균 수율 (%) | 총 생산량 / 총 투입량 × 100 | 주황 `#f89c1e` |
| 도정 건수 | `MillingBatch` 레코드 수 | 회색 `#94a3b8` |

### 차트
- 투입량/생산량: 겹침 막대 (투입=연한 배경, 생산=진한 전경, 동일 폭)
- 수율: 주황 라인 (우측 Y축 50~100%)
- X축 레이블: groupBy에 따라 일/주/월 포맷 자동 전환

### 하단 테이블
- `MillingBatch` 기준 행, 클릭 시 인라인 상세 표시
- 컬럼: 날짜 | 도정종류 | 투입량(kg) | 생산량(kg) | 수율(%) | 품종 | 비고

### 개발 상태
- `[x]` 기본 구조 구현 완료 (2026-03-24)
- `[ ]` 필터 고도화 (빠른기간 칩 + 보조 필터 + groupBy 자동집계)

## 페이지 2: 농가/재고 현황

- **경로:** `/statistics/stock`
- **상태:** `[ ]` 기획 / `[ ]` 개발 / `[ ]` 완료

### 사용 데이터
- `Stock` + `Farmer` + `ProducerGroup`

### 필터
| 구분 | 항목 |
|------|------|
| 공통 필터 | 조회 기간, 작목반 |
| 보기 기준 탭 | 농가별 / 작목반별 / 품종별 |

### 요약 카드
- 총 입고량 (kg)
- 도정 완료량 (kg)
- 미처리 재고 (kg)
- 미처리 농가 수

### 차트
- 농가별 입고 → 처리 현황 가로 막대 (스택형)
- 재고 상태 비율 파이차트 (AVAILABLE / CONSUMED / RELEASED)

### 테이블
- 농가별 입고량 / 처리량 / 잔량
- 장기 미처리 재고 경고 (입고 후 N일 이상 AVAILABLE인 항목 강조)

## 페이지 3: 포장/출고 현황

- **경로:** `/statistics/output`
- **상태:** `[ ]` 기획 / `[ ]` 개발 / `[ ]` 완료

### 사용 데이터
- `MillingOutputPackage` + `StockRelease`

### 필터
| 구분 | 항목 |
|------|------|
| 공통 필터 | 조회 기간 |
| 보기 기준 탭 | 규격별 / 월별 / 출고처별 |

### 요약 카드
- 총 생산량 (kg)
- 총 출고량 (kg)
- 포장 총 수량 (개)
- 출고처 수

### 차트
- 포장 규격별 비율 파이차트 (20kg / 10kg / 5kg / 톤백)
- 월별 생산 / 출고 추이 비교 차트

### 테이블
- 규격별 생산 수량 / 중량
- 출고처별 출고량

## 개발 순서 (우선순위)

| 순서 | 페이지 | 이유 |
|-----|--------|------|
| 1 | 도정 실적 분석 | 수율이 핵심 KPI, 즉시 활용 가능 |

## 기술 스택

| 패키지 | 용도 | 상태 |
|--------|------|------|
| `recharts` | 꺾은선 / 막대 / 파이 차트 | ✅ 설치됨 |
| `@tanstack/react-table` | 전문 데이터 그리드 (정렬/필터/페이지네이션) | ✅ 설치됨 |
| `xlsx` | 엑셀 내보내기 | ✅ 설치됨 |

- **API:** Next.js Route Handler (`/api/statistics/...`)
- **데이터 집계:** Prisma groupBy 또는 raw query 활용
- **차트 스타일:** Tailwind 색상 변수 그대로 Recharts에 적용
- **엑셀 내보내기:** TanStack Table 데이터 → xlsx 변환 버튼 제공
